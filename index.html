About the Zoo Animal Dataset

This project uses the famous Zoo Animal Classification dataset, which contains information about various animals and their characteristics.

Dataset Features

The dataset contains 16 features for each animal:

  • Physical: hair, feathers, eggs, milk, backbone, fins, tail
  • Behavioral: airborne, aquatic, predator, breathes
  • Structural: toothed, venomous, legs, domestic, catsize
